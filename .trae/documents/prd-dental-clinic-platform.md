## 1. 产品概述

连锁口腔诊所集团运营管理平台，为多门店口腔医疗机构提供一站式数字化运营解决方案。涵盖预约、排班、物资、会员四大核心业务模块，实现各门店数据统一管理、高效协同。

- 目标用户：诊所运营管理者、前台接待、医生、护士、物资管理员
- 核心价值：提升运营效率、优化资源配置、降低管理成本、增强患者体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 集团管理员 | 账号密码 | 全系统管理、跨门店数据查看、系统配置 |
| 门店管理员 | 账号密码 | 本门店运营管理、人员排班、数据统计 |
| 医生 | 账号密码 | 查看个人排班、处理预约、查看患者信息 |
| 前台接待 | 账号密码 | 预约登记、会员管理、患者接待 |
| 物资管理员 | 账号密码 | 库存管理、采购入库、消耗出库 |

### 2.2 功能模块

1. **工作台首页**：数据概览、今日预约、待办事项、关键指标
2. **预约管理**：预约看板、预约列表、项目配置、预约登记
3. **排班管理**：排班日历、排班设置、冲突检测、排班统计
4. **物资管理**：库存总览、采购入库、消耗出库、一物一码追溯、库存预警
5. **会员管理**：会员档案、储值卡管理、分期付款、积分管理
6. **数据统计**：经营报表、预约分析、物资消耗、会员分析
7. **系统设置**：门店管理、人员管理、项目配置、权限管理

## 3. 核心流程

### 3.1 预约流程
患者到店/电话咨询 → 前台查询可用医生时段 → 选择治疗项目（自动计算时长）→ 关联患者信息 → 创建预约 → 系统自动关联排班 → 发送预约提醒

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：医疗蓝系渐变（#1E6FDB 至 #3B82F6），传递专业、可信的医疗形象
- **辅助色**：清新薄荷绿（#10B981）用于成功状态，暖橙（#F59E0B）用于预警提示，玫红（#EF4444）用于错误状态
- **中性色**：浅灰（#F8FAFC）背景，深灰（#1E293B）文字层次分明
- **圆角**：统一使用 8px 圆角，卡片采用 12px 圆角，传递友好现代感
- **布局风格**：左侧导航栏 + 顶部面包屑 + 内容卡片式布局
- **图标风格**：线性图标，2px 线条宽度，与医疗行业专业气质匹配

### 4.2 响应式设计
- **设计策略**：桌面端优先（1440px基准），自适应平板（768px），适配移动端（375px）
- **导航栏**：桌面端展开侧边栏，平板端折叠为图标，移动端变为底部Tab栏
- **数据表格**：桌面端完整显示，平板端隐藏次要列，移动端转为卡片列表展示
